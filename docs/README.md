## 改造後について
このアプリには､ 設定アプリから **DchaState** を変更できる機能があります｡  

<kbd>アプリと通知</kbd>  
→<kbd>XX個のアプリをすべて表示</kbd>  
→<kbd>DchaState Changer</kbd>  
→<kbd>詳細設定</kbd>  
→<kbd>アプリ内のその他の設定</kbd>

この｢<kbd><b>アプリ内のその他の設定</b></kbd>｣を押す度に値が変わります｡
